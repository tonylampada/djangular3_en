Djangular

This is a sample web application build with Django and AngularJS.

The app is an SPA (single page application) with a few desirable "architectural features", like

  • Project's executable help
  • front-end only development mode, with faked back-end
  • AngularJS component catalog
  • Model-as-angular-service pattern sample
  • Javascript unit tests
  • Some other good practices here and there

Enjoy :-)
